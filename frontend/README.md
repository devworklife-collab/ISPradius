# ISPradius Frontend

This folder is reserved for the administrative UI and dashboards.

The frontend should provide:

- Authentication and user management
- Unified admin dashboard
- Customer management screens
- Billing and invoice views
- Network device dashboards
- Ticketing and notifications
- Monitoring and analytics dashboards

## Recommended Stack

- React + TypeScript
- Tailwind CSS or UI library (e.g., Material UI)
- Charting (Recharts, Chart.js)
- API clients using `axios` or `fetch`
